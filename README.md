# MyCryptoPortfolio
ASP.NET Core 6 Training Application

# Getting Started
## Clone Repository
`git clone git@github.com:holmesdev/MyCryptoPortfolio.git`
## Restore Packages
`dotnet restore`

## Build Project
`dotnet build`

## Run Project
`dotnet run`  
Open browser to displayed URL + "/swagger" (i.e. https://localhost:7197/swagger)
